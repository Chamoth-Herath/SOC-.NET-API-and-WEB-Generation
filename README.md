# Event Management System (.NET Framework)

## Overview
A centralized event management system that provides APIs for event creation, participant registration, and organizer dashboards.

The system helps organizers efficiently manage events while allowing users to easily register and participate in events through a structured web platform.

## Features
- Event creation and management (Organizer)
- Participant registration system
- Organizer dashboard for event monitoring
- API-based architecture for data handling
- Dynamic web interface using Razor Pages
- Event and user registration tracking

## Technologies Used
- .NET Framework
- Razor Pages
- C# (Backend logic)
- RESTful APIs
- SQL Server (Database)

## Testing & Quality Assurance
- Functional testing of APIs and web modules
- UI testing for dashboards and user interfaces
- Data integrity validation
- Bug reporting for:
  - Event management module
  - User registration system
  - API response validation
- End-to-end workflow testing

## Project Type
Web Application / API-based Event Management System
